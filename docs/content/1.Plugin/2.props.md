# Modal props

#### All modal props

| **Prop** | **Type** | **Default** | **Description**                                       |
|----------|----------|-------------|-------------------------------------------------------|
| `title`  | `string` |             | Modal title                                           |
| `width`  | `number` | 400         | Modal width value                                     |
| `height` | `number` | 350         | Modal height value                                    |
